## 설치 및 설정

### 1. Python 설치 확인
```powershell
python --version
```
Python 3.8 이상이 필요합니다.

### 2. 필요한 패키지 설치
```powershell
pip install -r requirements.txt
```

설치되는 패키지:
- `selenium` - 웹 자동화
- `webdriver-manager` - Chrome 드라이버 자동 관리
- `customtkinter` - 현대적인 GUI
- `google-generativeai` - Gemini AI 연동

## Gemini API 키 발급

### 1. Google AI Studio 접속
- URL: https://aistudio.google.com/
- Google 계정으로 로그인

### 2. API 키 생성
1. 좌측 메뉴에서 "Get API Key" 클릭
2. "Create API Key" 버튼 클릭
3. 프로젝트 선택 또는 새로 생성
4. API 키 복사 (나중에 다시 확인 가능)

### 3. API 키 보관
- API 키는 안전하게 보관하세요
- 타인과 공유하지 마세요
- 무료 사용량: 월 60회 요청

## 포스팅 프로세스

### 자동화 순서

1. **브라우저 실행**
   - Chrome 브라우저 자동 실행
   - 웹드라이버 자동 설정

2. **AI 글 생성**
   - Gemini AI가 입력한 주제로 글 작성
   - 제목과 본문 자동 생성
   - 1000~2000자 분량

3. **네이버 로그인**
   - naver.com 접속
   - 로그인 버튼 클릭
   - 자동 로그인 처리

4. **블로그 페이지 이동**
   - 블로그 홈 페이지 접속
   - 글쓰기 버튼 클릭

5. **글 작성**
   - iframe 전환
   - 제목 입력
   - 본문 입력

6. **발행 설정**
   - 주제 선택 (선택사항)
   - 공개 설정
   - 태그 입력

7. **발행**
   - 발행 버튼 클릭
   - 최종 발행 확인

## FAQ

### Q1. Gemini API 키가 무료인가요?
**A:** 네, 무료로 사용 가능합니다. 월 60회 요청 한도가 있으며, 일반적인 블로그 포스팅에는 충분합니다.

### Q2. 글이 마음에 들지 않으면 어떻게 하나요?
**A:** `prompt.txt` 파일을 수정하여 AI의 작성 스타일을 변경할 수 있습니다. 더 구체적인 요구사항을 추가하세요.

### Q3. 한 번에 여러 개의 글을 포스팅할 수 있나요?
**A:** 현재는 한 번에 하나의 글만 포스팅됩니다. 반복 포스팅 기능은 추후 업데이트 예정입니다.

### Q4. 이미지도 자동으로 추가되나요?
**A:** 현재 버전은 텍스트만 지원합니다. 이미지 업로드 기능은 추후 업데이트 예정입니다.

### Q5. 네이버가 자동화를 감지하지 않나요?
**A:** 다음 방법으로 감지를 최소화합니다:
- User-Agent 설정
- 자동화 플래그 제거
- 적절한 대기 시간 설정
- JavaScript를 통한 입력

### Q6. 예약 포스팅이 가능한가요?
**A:** 현재 버전은 즉시 포스팅만 지원합니다. 예약 기능은 추후 업데이트 예정입니다.

### Q7. 카테고리 설정은 어떻게 하나요?
**A:** 현재는 블로그 주제만 설정 가능합니다. 카테고리는 네이버 블로그에서 기본 카테고리로 설정됩니다.

### Q8. 에러가 발생하면 어떻게 하나요?
**A:** 
1. 상태 메시지를 확인하세요
2. 브라우저 화면에서 어디서 멈췄는지 확인하세요
3. README.md의 "문제 해결" 섹션을 참고하세요
4. 필요시 `automation.py`의 `time.sleep()` 값을 늘려보세요

### Q9. 크롬이 아닌 다른 브라우저를 사용할 수 있나요?
**A:** 현재는 Chrome만 지원합니다. Firefox나 Edge 지원은 추후 업데이트 예정입니다.

### Q10. API 키를 매번 입력해야 하나요?
**A:** 현재는 매번 입력해야 합니다. 설정 저장 기능은 추후 업데이트 예정입니다.

## 💡 팁

### 1. 효과적인 주제 입력
❌ 나쁜 예: "파이썬"
✅ 좋은 예: "파이썬 초보자를 위한 변수와 데이터 타입 완벽 가이드"

### 2. prompt.txt 커스터마이징
기본 프롬프트를 수정하여 원하는 스타일의 글을 생성할 수 있습니다:
```
블로그에 게시할 글을 작성해주세요.

주제: {topic}

다음 사항을 고려하여 작성해주세요:
- 전문적이고 신뢰할 수 있는 내용
- 독자들이 실제로 활용할 수 있는 팁
- SEO를 고려한 키워드 사용
- 적절한 소제목 구성
- 1500자 이상의 충실한 내용
```

### 3. 태그 활용
관련성 높은 태그를 3~5개 입력하면 검색 노출에 도움이 됩니다.
예: `파이썬, 프로그래밍, 코딩, 개발, 초보자가이드`

### 4. 공개 설정
- **전체공개**: 모든 사람이 볼 수 있음 (추천)
- **이웃공개**: 이웃만 볼 수 있음
- **서로이웃공개**: 서로이웃만 볼 수 있음
- **비공개**: 본인만 볼 수 있음 (테스트용)
