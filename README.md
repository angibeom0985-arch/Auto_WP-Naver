# 네이버 블로그 AI 자동 포스팅 프로그램

**Gemini AI**를 활용하여 자동으로 블로그 글을 생성하고, **Selenium**으로 네이버 블로그에 자동 포스팅하는 데스크톱 애플리케이션입니다.

## 🎯 주요 기능

- 🤖 **Gemini AI 글 자동 생성** - 주제만 입력하면 AI가 전문적인 블로그 글 작성
- ✅ 네이버 자동 로그인
- ✅ 블로그 글 자동 작성 및 발행
- ✅ 주제 설정 (40개 이상의 카테고리)
- ✅ 공개 설정 (전체공개/이웃공개/서로이웃공개/비공개)
- ✅ 태그 자동 입력
- ✅ 네이버 스타일의 직관적인 GUI
- ✅ 실시간 상태 표시
- ✅ 비동기 처리로 GUI 응답성 유지

## 📋 요구사항

- Python 3.8 이상
- Chrome 브라우저

## 🚀 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

## 💻 사용 방법

### 1. Gemini API 키 발급
1. https://aistudio.google.com/ 접속
2. Google 계정으로 로그인
3. "Get API Key" 클릭하여 API 키 발급

### 2. keywords.txt 파일 설정
1. `keywords.txt` 파일을 열어 키워드 입력
2. 한 줄에 하나씩 입력 (예: 파이썬, 프로그래밍, 코딩)
3. `#`으로 시작하는 줄은 주석으로 처리됩니다

### 3. 프로그램 실행
```bash
python app_gui.py
```

### 4. GUI에서 정보 입력
1. **Gemini API 키** 입력
2. **네이버 로그인 정보** 입력:
   - 네이버 아이디
   - 비밀번호
3. **AI 글 생성 설정**:
   - 글 주제 입력 (예: "파이썬 초보자를 위한 가이드")
   - 키워드는 keywords.txt에서 자동으로 불러옵니다
4. **주제 설정** (선택사항):
   - 40개 이상의 블로그 주제 중 선택 (라디오 버튼)
   - 또는 "주제 선택 안 함" 선택
5. **포스팅 설정**:
   - 공개 설정 선택 (전체공개/이웃공개/서로이웃공개/비공개)
6. **외부 링크** (선택사항):
   - 링크 URL 입력
   - 링크 텍스트 입력

### 5. "AI 포스팅 실행" 버튼 클릭
- keywords.txt의 키워드를 기반으로 AI가 글을 생성합니다
- 자동으로 네이버 블로그에 포스팅됩니다
- 태그는 keywords.txt의 키워드로 자동 생성됩니다
- 브라우저에서 자동으로 진행되는 과정을 볼 수 있습니다

## ⚠️ 중요 안내

### 실제 네이버 블로그 셀렉터 적용됨
`automation.py` 파일에는 실제 네이버 블로그의 셀렉터가 이미 적용되어 있습니다:

- ✅ 로그인 버튼: `.MyView-module__link_login___HpHMW`
- ✅ 아이디/비밀번호: `.input_item.id`, `.input_item.pw`
- ✅ 글쓰기 버튼: `.sp_common.icon_write`
- ✅ 제목 입력: `.se-section.se-section-documentTitle`
- ✅ 본문 입력: `.se-section.se-section-text`
- ✅ 발행 버튼: `.publish_btn__m9KHH`
- ✅ 주제 설정: `.set_theme__KLbnU`
- ✅ 공개 설정: `#open_public`, `#open_neighbor`, 등
- ✅ 태그 입력: `.tag_textarea__CD7pC`

### 네이버 블로그 구조
- 블로그 에디터는 **iframe**으로 감싸져 있습니다
- 자동화 코드에서 iframe 전환을 자동으로 처리합니다

## 🔧 문제 해결

### 브라우저가 실행되지 않는 경우
- Chrome 브라우저가 설치되어 있는지 확인
- `webdriver-manager`가 자동으로 드라이버를 다운로드합니다
- 인터넷 연결 확인

### 로그인이 실패하는 경우
- 아이디/비밀번호 확인
- 네이버 보안 설정 확인 (캡차, 2단계 인증 등)
- 로봇 감지를 피하기 위해 속도를 조절하는 `time.sleep()` 추가 가능

### AI 글 생성이 실패하는 경우
- Gemini API 키가 올바른지 확인
- API 키의 사용 한도 확인
- 인터넷 연결 확인
- `prompt.txt` 파일이 있는지 확인

### 요소를 찾을 수 없다는 오류
- 네이버 블로그 구조가 변경되었을 수 있습니다
- 개발자 도구(F12)로 현재 셀렉터 확인 후 `automation.py` 수정

### `google.generativeai` 모듈을 찾을 수 없음
```bash
pip install google-generativeai
```
